# My Cinema Lister

### What is My Cinema Lister?
My Cinema Lister is a simple web site made with Flask/Python that allows users to register/log in and review their favourite movies. This web site was made as a school project.


### What dois it include?
1. Front page (guest):
```
- Login and Register options
- Possible to see all reviews that are created by other people
```

2. Front page (for user that is logged in)
```
- Search for your movie through an API, write a review for the movie and give it a rating (1-10). Add it to your reviewed movies list.
- View all of your reviewed movies that you can edit and or delete.
```

### Where do we fetch the movies from?

- All of the movies that you search with the search bar are fetched from a public API here: http://www.omdbapi.com/
